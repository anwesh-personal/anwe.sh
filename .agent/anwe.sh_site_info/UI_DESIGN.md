## Component Library

Use these components consistently:

- **Card**: Rounded corners, subtle shadow, hover effect
- **Button Primary**: Solid background, white text
- **Button Secondary**: Border only, colored text
- **Button Danger**: Red background for destructive actions
- **Input**: Full width, border, focus ring
- **Toggle**: Switch for on/off settings
- **Modal**: Centered, overlay backdrop, close button
- **Tabs**: Underline active, gray inactive
