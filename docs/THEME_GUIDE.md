## ğŸ¨ Color Usage Guidelines

### Primary Colors
- Use for main actions, links, and important elements
- High contrast with background

### Secondary Colors
- Use for secondary actions and accents
- Complements primary color

### Background Colors
- **Background**: Main page background
- **Background Secondary**: Cards, panels
- **Background Tertiary**: Hover states, nested elements

### Foreground Colors
- **Foreground**: Main text
- **Foreground Secondary**: Secondary text, labels

### Semantic Colors
- **Success**: Success messages, positive actions
- **Warning**: Warnings, caution states
- **Error**: Errors, destructive actions
- **Info**: Informational messages

## ğŸ”„ Dark Mode Implementation

All brand kits support dark mode with:
- Inverted color schemes
- Adjusted contrast ratios
- Glow effects (where appropriate)
- Smooth transitions

### Dark Mode Best Practices
1. Test contrast ratios (WCAG AA minimum)
2. Adjust shadows for dark backgrounds
3. Use subtle glows for tech themes
4. Maintain brand identity in both modes
