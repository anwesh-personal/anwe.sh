‚ûï Adding New Agents

The system is designed to be extensible. To add a new agent:

1. Register in Database

INSERT INTO agents (name, type, description, model, config)
VALUES (
  'translator',
  'content',
  'Translates content to other languages',
  'gpt-4-turbo',
  '{
    "languages": ["Spanish", "French", "Hindi"],
    "preserve_tone": true
  }'
);

2. Create Agent Handler

// backend/agents/translator.js
export class TranslatorAgent {
  constructor(config) {
    this.config = config;
    this.model = config.model;
  }

  async execute(input) {
    const { content, targetLanguage } = input;
    
    const prompt = `
      Translate the following content to ${targetLanguage}.
      Preserve the tone and style of the original.
      
      Content:
      ${content}
    `;
    
    const result = await this.llm.complete(prompt);
    
    return {
      translatedContent: result,
      sourceLanguage: 'English',
      targetLanguage
    };
  }
}

3. Register with ORA

// backend/agents/ora.js
import { TranslatorAgent } from './translator.js';

// Add to ORA's agent registry
ora.registerAgent('translator', TranslatorAgent);

// Now ORA can use it:
// "@ora translate my latest post to Spanish"
