blog_posts

Stores all blog posts, both published and drafts. Supports AI-generated content tracking.

Column Type Constraints Description
id UUID PRIMARY KEY, DEFAULT uuid_generate_v4() Unique identifier
title TEXT NOT NULL Post title
slug TEXT NOT NULL, UNIQUE URL-friendly identifier
excerpt TEXT nullable Short summary for cards
content TEXT NOT NULL Full post content (Markdown)
cover_image TEXT nullable URL to cover image
category TEXT NOT NULL, DEFAULT 'General' Post category
tags TEXT[] DEFAULT '{}' Array of tag strings
reading_time TEXT nullable Estimated read time (e.g., "5 min")
word_count INTEGER nullable Total word count
published BOOLEAN DEFAULT FALSE Is post visible?
published_at TIMESTAMPTZ nullable When post went live
featured BOOLEAN DEFAULT FALSE Show in featured section?
meta_title TEXT nullable SEO title override
meta_description TEXT nullable SEO description
author_name TEXT DEFAULT 'Anwesh Rath' Display author name
source TEXT DEFAULT 'manual' 'manual', 'ora', 'agent'
source_agent TEXT nullable Which agent created it
source_prompt TEXT nullable Original AI prompt
created_at TIMESTAMPTZ DEFAULT NOW() Record creation time
updated_at TIMESTAMPTZ DEFAULT NOW() Last modification time

Indexes

CREATE INDEX idx_blog_posts_slug ON blog_posts(slug);
CREATE INDEX idx_blog_posts_published ON blog_posts(published, published_at DESC);
CREATE INDEX idx_blog_posts_category ON blog_posts(category);
CREATE INDEX idx_blog_posts_featured ON blog_posts(featured) WHERE featured = TRUE;

Row Level Security

Public can read published posts

SELECT where published = TRUE

Service role has full access

ALL where auth.role() = 'service_role'

users

Admin users who can access the dashboard and manage content.

Column Type Constraints Description
id UUID PRIMARY KEY Unique identifier
email TEXT NOT NULL, UNIQUE Login email
password_hash TEXT NOT NULL Bcrypt hashed password
name TEXT NOT NULL Display name
role TEXT DEFAULT 'admin' 'superadmin', 'admin', 'editor'
avatar_url TEXT nullable Profile picture URL
created_at TIMESTAMPTZ DEFAULT NOW() Account creation
last_login TIMESTAMPTZ nullable Last login time

agents

Registry of all AI agents in the system.

Column Type Constraints Description
id UUID PRIMARY KEY Unique identifier
name TEXT NOT NULL, UNIQUE Agent name (e.g., 'writer')
type TEXT NOT NULL 'content', 'image', 'seo', 'social'
description TEXT nullable What this agent does
active BOOLEAN DEFAULT TRUE Is agent enabled?
config JSONB DEFAULT '{}' Agent configuration
model TEXT nullable AI model used (gpt-4, claude-3)
last_run TIMESTAMPTZ nullable Last execution time
run_count INTEGER DEFAULT 0 Total executions
created_at TIMESTAMPTZ DEFAULT NOW() Registration time

agent_logs

Audit trail of all agent executions.

Column Type Constraints Description
id UUID PRIMARY KEY Unique identifier
agent_id UUID REFERENCES agents(id) Which agent ran
triggered_by UUID REFERENCES users(id) Who triggered it
trigger_type TEXT NOT NULL 'manual', 'scheduled', 'webhook', 'agent'
action TEXT NOT NULL What action was performed
status TEXT NOT NULL 'pending', 'running', 'success', 'failed'
input JSONB nullable Input parameters
output JSONB nullable Result data
error TEXT nullable Error message if failed
duration_ms INTEGER nullable Execution time in ms
tokens_used INTEGER nullable AI tokens consumed
cost_usd DECIMAL(10,4) nullable Estimated cost
created_at TIMESTAMPTZ DEFAULT NOW() Execution start time

analytics_events

Raw analytics events for tracking visitor behavior.

Column Type Constraints Description
id UUID PRIMARY KEY Unique identifier
post_id UUID REFERENCES blog_posts(id) Related post (if any)
event_type TEXT NOT NULL 'pageview', 'click', 'scroll', 'share'
page_path TEXT NOT NULL URL path visited
referrer TEXT nullable Traffic source
user_agent TEXT nullable Browser/device info
country TEXT nullable Visitor country
session_id TEXT nullable Anonymous session ID
metadata JSONB DEFAULT '{}' Extra event data
created_at TIMESTAMPTZ DEFAULT NOW() Event timestamp

comments

User comments on blog posts (optional feature).

Column Type Constraints Description
id UUID PRIMARY KEY Unique identifier
post_id UUID REFERENCES blog_posts(id) Parent post
author_name TEXT NOT NULL Commenter name
author_email TEXT NOT NULL Commenter email
content TEXT NOT NULL Comment text
approved BOOLEAN DEFAULT FALSE Moderation status
parent_id UUID REFERENCES comments(id) For nested replies
created_at TIMESTAMPTZ DEFAULT NOW() Comment time

analytics_events INSERT only SELECT only Full access
comments SELECT (approved), INSERT Full CRUD Full access
