üìä Workflow Triggers

MANUAL Admin Panel Actions

Direct actions from the admin dashboard:

  • Admin logs into app.anwe.sh
  • Navigates to Blog ‚Üí New Post
  • Writes or pastes content in rich editor
  • Optionally clicks "Enhance with AI"
  • Reviews preview and publishes
  • Post goes live within 60 seconds (ISR)

Error Categories

  • Recoverable: API timeouts, rate limits, temporary failures
  • Critical: Database down, auth failure, agent crash
  • User Error: Invalid input, permission denied
